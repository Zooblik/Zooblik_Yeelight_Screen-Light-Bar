2) ПОДКЛЮЧЕНИЕ ESP к USB-TTL (смотри распиновку на картинке connexion.jpg)
ВНИМАНИЕ! не подключайте 3.3V ESP к USB-TTL так как лампа включится и светодиоды начнут потреблять серьезный ток через USB-TTL, что может привести к серьезным проблемам со стороны USB-TTL или USB порта компьютера!
Подключать так можно, только при условии что во время подключения питания к ESP, контакт GPIO_0 (GPIO0) подключен к земле - тогда ESP загружается в режиме прошивке, а не в рабочем режиме (светодиоды не запустятся)

- tx (ESP) -> rx (USB-TTL)
- rx (ESP) -> tx (USB-TTL)
- gnd (ESP) -> gnd (USB-TTL)
- 3,3V (ESP) -> 3,3V (USB-TTL) - Лично я запитывал от Лабораторного блока управления (ЛБП)

3) ПРОШИВКА ESP
ВНИМАНИЕ! Чтобы прошивка прошла успешно, ESP нужно перевести в режим прошивки.
Для этого перед подачей питания на ESP ОБЯЗАТЕЛЬНО нужно замкнуть GPIO_0 на землю (GND).
Замкнуть нужно только во время включения, после включения нужно разомкнуть контакт.

- Установите Python на компьютер https://www.python.org/downloads
- Установите ESPHome
pip install esphome

- Скачайте проект с GitHub к себе на компьютер и разорхивируйте проект.
ВНИМАНИЕ! В файле main.yaml нужно указать данные для WIFI
wifi:
ssid: "MyFiWi" # Введите Имя Вашей WiFi сети (в кавычках)
password: "12345678" # Введите пароль от WiFi (в кавычках)

- В терминале перейдите в папку скаченного проекта и запустите команду
python -m esphome run main.yaml

Начнется подготовка к прошивке, начнется скачиваение необходимых программ, библиотек - это займет некоторое время. Если у вас несколько COM устройств на компьютере, то терминал попросит у Вас выбрать конкретное устройство (нужно выбрать порт с USB-TTL)

В случае успешной прошивки увидете надпись
INFO Successfully uploaded program.
INFO Starting log output from COM5 with baud rate 115200

После чего можно проверять результат, включить лампу с пульта, проверить работу пульта.
